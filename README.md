# GeoJupyter initiatives

This repository acts as a "roadmap" for the GeoJupyter community.

The reason this repository exists is to enable roadmapping as a living discussion, not a
static artifact.


## Information management

The roadmap is managed as GitHub issues.


### Labels

Labels are used to categorize and prioritize the issues.


### Issue structure

This repository includes a "linter" (in the `validator/` directory) for those issues
which ensures each issue provides necessary details for us to effectively plan each
initiative.


### Project board

The
[GeoJupyter Initiatives Project Board](https://github.com/orgs/geojupyter/projects/6)
gives us an interface to visualize issues.
This is a work in progress.


## Acknowledgements

This is inspired by the
[2i2c initiatives repository](https://github.com/2i2c-org/initiatives)
and the JupyterHub community's roadmapping workshop in early 2026.
